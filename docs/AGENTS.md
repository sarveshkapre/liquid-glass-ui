# AGENTS

## Purpose
Liquid Glass UI is a reference component showcase for frosted, Apple-inspired surfaces. Keep the site fast, accessible, and easy to clone.

## Guardrails
- No authentication or tracking.
- Prefer tiny dependencies and CSS-first styling.
- Keep all tokens and component examples in sync.

## Commands
- Setup: `make setup`
- Dev server: `make dev`
- Lint: `make lint`
- Typecheck: `make typecheck`
- Test: `make test`
- Build: `make build`
- Quality gate: `make check`

## Release
Follow `docs/RELEASE.md` and update `docs/CHANGELOG.md` with user-facing changes.
